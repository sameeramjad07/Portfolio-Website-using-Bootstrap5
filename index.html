About Me

Hi! This is Sameer Amjad, A Web Developer and a Computer Scientist at NUST-SEECS.
Im a T3 Stack Developer.
The T3 Stack is a web development stack designed for simplicity, modularity, and full-stack type safety. It's a collection of tools that work together seamlessly to create efficient and robust web applications.

Here's a breakdown of the core components:

1. Frontend:

Next.js

A popular React framework used for building performant and scalable web applications.

TypeScript

A superset of JavaScript that adds optional static typing, improving code clarity and catching errors early in the development process.

Tailwind CSS

A utility-first CSS framework offering rapid styling with pre-built classes.

2. Backend:

Prisma

A powerful ORM (Object-Relational Mapper) that simplifies interacting with databases using TypeScript.

tRPC

A type-safe RPC (Remote Procedure Call) framework for creating APIs with clear interfaces and error handling.

NextAuth.js

An authentication solution for Next.js applications, providing features like social login and JWT authentication.
